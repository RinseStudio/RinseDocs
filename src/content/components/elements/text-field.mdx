## Basic usage

The following is an example of a basic textfield.

<RinseUI2 url="/components/text-field/basic-text-field" isPreviewOnly />

```mdx
import { TextField } from '@/components/TextField';

<TextField type="email" value="jane@company.com" />
```

## Text field sizes

Text fields sizes can be changed with the `size` property.

<RinseUI2 url="/components/text-field/text-field-sizes" isPreviewOnly />

```mdx
import { TextField } from '@/components/TextField';

<TextField size="xs" type="email" value="jane@company.com" />
<TextField size="sm" type="email" value="jane@company.com" />
<TextField size="md" type="email" value="jane@company.com" />
<TextField size="lg" type="email" value="jane@company.com" />
<TextField size="xl" type="email" value="jane@company.com" />
```

## Validations

Text fields can have validation states.

<RinseUI2 url="/components/text-field/text-field-validations" isPreviewOnly />

```mdx
import { TextField } from '@/components/TextField';

<TextField placeholder="First name" />
<TextField placeholder="First name" validation="success" />
<TextField placeholder="First name" validation="error" />
```

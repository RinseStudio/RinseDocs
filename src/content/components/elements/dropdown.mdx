## API reference

The following props and prop values are available for this component.

| Prop       | Values                                     | Default   |
| ---------- | ------------------------------------------ | --------- |
| `size`     | `xs`, `sm`, `md`, `lg`, `xl`               | `md`      |
| `variant`  | `primary`, `secondary`, `outline`, `ghost` | `primary` |
| `isDanger` | `boolean`                                  | `false`   |
| `as`       | `button`, `a`, `div`                       | `button`  |
